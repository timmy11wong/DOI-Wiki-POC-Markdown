# DOI Wiki POC Markdown repository

This repository is intended as a POC and testing ground for documenting content using Markdown format on GitHub. It is organized as a wiki, containing only Markdown files and related images. The sample data presented here is based on the game records from the D&D 5R campaign "Dream of Immortal" (DOI).

## Scope

The scope of this repository includes the following aspects:

- Support for multiple collaborators contributing to the repository
- Publicly accessible content for viewing
- Wiki-style hyperlinks and illustrations
- Free hosting on GitHub
- Minimal maintenance requirements

## Getting Started

To get started, you can clone this repository to your local machine using Git. Once cloned, you may open the folder with Obsidian or any other note-taking application that supports Markdown format.

**Steps:**

1. Clone the Repository

    Open your terminal and run:

    ```bash
    git clone https://github.com/timmy11wong/DOI-Wiki-POC-Markdown.git
    ```

2. Open with Obsidian or Other Markdown Applications

    - Launch Obsidian and choose “Open folder as vault,” then select the cloned repository folder.
    - Alternatively, you can open the folder with any Markdown-compatible notes app to view and edit the documents.
